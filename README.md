
# Aeonaxy React Tailwind

Demo: https://aeonaxy-react-tailwind-ritik.netlify.app/


## Run Locally

Clone the project

```bash
  git clone https://github.com/RitikKumar202/AeonaxyReactTailwindRitik.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Tech Stack

React, Html, Tailwind CSS
